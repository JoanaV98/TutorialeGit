Descripción de comandos Git





git-init: crea un repositorio de Git vacío o reinicializa uno existente


git-status: es para enumerar los archivos que se han preparado, los que están sin preparar y los archivos sin seguimiento.

git-add: para agregar el contenido del archivo al índice


git-commit: para agregar el contenido del archivo al índice, y registrar cambios en el repositorio

git-checkout: crea ramas y te ayuda a navegar entre ellaso para cambiar de rama o restaurar

git restore: es una herramienta para revertir cambios no comprometidos. Los cambios no comprometidos son: a) cambios en su copia de trabajo, o b) contenido en su índice (también conocido como área de preparación).


git-diff: De forma predeterminada, git diff mostrará todos los cambios sin confirmar desde la última confirmación.


git-branch: Enumerar, crear o eliminar sucursales


git-branch: Enumerar, crear o eliminar sucursales


git-clone: Clonar un repositorio en un directorio nuevo


git-log: El comando git log muestra las instantáneas confirmadas. Te permite ver el historial del proyecto, filtrarlo y buscar cambios concretos. Mientras que git status te permite examinar el directorio de trabajo y el entorno de ensayo, git log solo muestra el historial confirmado.

git-fetch: descargará el contenido remoto sin modificar el estado del repositorio local.


git-merge: permite tomar las líneas independientes de desarrollo creadas por git branch e integrarlas en una sola rama. ... Una vez más, esto significa que git merge se suele utilizar junto con git checkout para seleccionar la rama actual y git branch -d para eliminar la rama de destino obsoleta.


git push: permite subir los commits desde la rama (branch) local en tu repositorio git local al repositorio remoto.
git pull: descargará el contenido remoto y tratará inmediatamente de cambiar el estado del repositorio local para reflejar ese contenido.


git-rm: El comando solo elimina rutas confirmadas actualmente en el repositorio de Git.


git-switch: cambiar de rama


git-remote: es una manera más simple de enviar URL a estos comandos de "compartición".
